# CursoemVideoJava

Este repositório contém os exercícios do curso gratuito de Java focado em Programação Orientada a Objetos (POO) do site Curso em Vídeo, ministrado pelo professor Gustavo Guanabara.

O curso aborda os principais pilares da orientação a objetos em Java, incluindo abstração, encapsulamento, herança e polimorfismo, com exemplos práticos para fixação dos conceitos.

Neste repositório, estão organizados diversos arquivos .java divididos em pastas que correspondem às aulas e exercícios do curso. O objetivo é servir como um local de aprendizado e prática contínua, atualizando os códigos conforme o avanço no curso.

## Sobre o curso Curso em Vídeo - Java POO

- Curso gratuito e popular no Brasil para aprendizado de Java com foco em POO.
- Ministrado por Gustavo Guanabara, referência em ensino de programação.
- Abrange teoria e prática, desde conceitos básicos até projetos mais complexos.
- Uso do ambiente NetBeans para desenvolvimento e execução dos exercícios.

## Conteúdo atual do repositório

- Classes representando objetos do cotidiano, como Caneta, Celular, Teclado, entre outros.
- Métodos para manipulação de atributos e comportamento (exemplos: status, rabiscar, tampar).
- Conceitos de encapsulamento através de variáveis de instância e modificações de estado.
- Exercícios organizados em diferentes pacotes/baseados nas aulas.

## Objetivos do repositório

- Facilitar o aprendizado prático de Java POO.
- Manter uma coleção atualizada dos exercícios realizados no curso.
- Fornecer exemplos claros para referência futura e estudos.

Este repositório continuará sendo atualizado conforme o progresso no curso, servindo como um diário de aprendizado e desenvolvimento em Java POO.
